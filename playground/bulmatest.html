What is JS, and how does it fit with HTML and CSS?

Why three languages? Why not one?

In the beginning, there was HTML, which was used for all webpage content. CSS was developed in the mid-90s to separate styling from structure, and a year later, JavaScript was created for interactivity. As web development evolved, these three languages became the standard, supported by all browsers.

What is the DOM?

The Document Object Model (DOM) is a programming interface that allows programs to manipulate a document‚Äôs structure, style, and content. It is often referred to as a tree structure of nodes, where each node represents an HTML element that can be manipulated by JavaScript or other programming languages.

JavaScript stuff

Arrays vs Objects

Arrays

An array can contain different types of information like strings, numbers, and other arrays. A simple array might look like this:

let animals = ['zebra', 'flatworm', 'jellyfish', 'scallop'];

Objects

Objects are another way to store data with key-value pairs. Here is an example of an object:

const animalCounts = {
  zebra: 5,
  flatworm: 3.5,
  jellyfish: '???',
  scallop: 42
};

Control Flow and Loops

Control Flow

Control flow refers to the order in which individual statements, instructions, or function calls are executed in a program. Example:

  1. Start: Decide to make coffee.
  2. Check that you have coffee beans. If yes, proceed; if no, check for instant coffee, or cry if neither.
  3. ... (Steps continue)

Loops

Loops allow you to repeat a set of instructions until a certain condition is met. For example, emptying the dishwasher:

  1. Start: Open the dishwasher after the cycle ends.
  2. Check for items, remove, and repeat until empty.

Functions

Functions are reusable blocks of code designed to perform a specific task. Example of a function in pseudo-code:

function checkDishwasher() {
  if (dishwasher is closed) {
    openDishwasher();
  } else {
    removeItem();
  }
}

Functions are useful for code reusability, modularity, abstraction, and maintaining scope and encapsulation.
